# 데이터 마이그레이션 통합 엑셀 스펙

> 작성일: 2025-12-18  
> 목적: 3개 CSV 파일(통합데이터, 훈련과정목록, 자기부담금)을 하나의 통합 엑셀로 만들기 위한 컬럼 정의

---

## 배경

현재 데이터 마이그레이션 스크립트(`convert.py`)는 여러 개의 CSV 파일을 조인하여 처리하고 있습니다:

| 파일명 | 용도 | 조인 키 |
|--------|------|---------|
| 훈련과정목록-표 1.csv | 강좌 기본 정보 | 훈련과정명 |
| 통합데이터-표 1.csv | 카테고리 + 커리큘럼 레벨 매핑 | 훈련과정명 |
| 자기부담금-표 1.csv | 자비부담금 (lecture_fee) | 훈련과정명 |

스크립트 로직을 단순화하고 데이터 관리를 용이하게 하기 위해 **단일 통합 엑셀 파일**로 병합하는 것을 제안합니다.

## 스크립트 변경 시 장점

1. **단순화된 파싱**: `pd.read_csv(file, header=0)`로 간단히 로드
2. **조인 로직 제거**: 3개 파일 조인 → 단일 파일 읽기
3. **데이터 정합성**: 엑셀에서 직접 데이터 확인/수정 가능
4. **유지보수 용이**: 컬럼 추가/삭제 시 엑셀만 수정
