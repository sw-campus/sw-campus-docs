# SW Campus Docs 📚

SW Campus 프로젝트의 기술 문서, 코드 규칙, 개발 계획을 관리하는 저장소입니다.

---

## 📁 문서 구조

```
sw-campus-docs/
├── code-rules/          # 코드 규칙 및 컨벤션
├── prd/                 # 제품 요구사항 문서
├── plans/               # 개발 계획 및 스프린트
└── reports/             # 회의록, 회고, 리포트
```

| 저장소 | 설명 |
|--------|------|
| [sw-campus-server](https://github.com/sw-campus/sw-campus-server) | 백엔드 서버 |

## ✍️ 문서 작성 가이드

### 새 문서 추가 시

1. 적절한 폴더에 마크다운 파일 생성
2. 파일명은 `kebab-case` 사용

### 커밋 메시지 컨벤션

```
docs: 새 문서 추가
docs: 기존 문서 수정
docs: 오타 수정
```
