## 1. 개요

### 1.1 배경

> 이 기능이 필요한 이유와 비즈니스 컨텍스트를 설명합니다.

마이페이지에서 사용자가 수강 완료한 강의 목록(`/completed-lectures`)을 조회한 후, 특정 강의에 대해 본인이 작성한 후기의 상세 정보(카테고리별 점수 포함)를 조회할 수 있는 API가 필요합니다.

현재는 `/mypage/reviews`로 전체 후기 목록만 조회 가능하고, 특정 강의의 후기 상세를 조회하는 전용 API가 없습니다.

### 1.2 목적

> 이 기능이 달성하고자 하는 목표를 명확히 합니다.

- 수강 완료한 강의 목록에서 선택한 강의의 후기 상세 정보를 조회
- 카테고리별 점수(TEACHER, CURRICULUM, MANAGEMENT, FACILITY, PROJECT)를 포함한 상세 정보 제공
- 불필요한 `/mypage/reviews` API 정리

### 1.3 범위

**포함**:
- 특정 강의에 대한 내 후기 상세 조회 API
- `/mypage/reviews` API 삭제

**제외**:
- 다른 사용자의 후기 조회 (기존 `/api/v1/reviews/{reviewId}` 사용)
- 후기 작성/수정 기능 (기존 API 유지)

## 6. 제약사항 및 가정

### 6.1 제약사항

- 본인이 작성한 후기만 조회 가능
