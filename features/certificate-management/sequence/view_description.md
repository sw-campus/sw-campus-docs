# 수료증 이미지 조회 - 시퀀스 설명

## 개요

마이페이지 "내 후기" 탭에서 사용자가 업로드한 수료증 이미지를 확인하는 흐름입니다.

## 참여자 (Actors)

| Actor | 설명 |
|-------|------|
| User | 일반 사용자 |
| Client | 프론트엔드 (Next.js) |
| Server | 백엔드 API 서버 (Spring Boot) |
| DB | PostgreSQL 데이터베이스 |

## 흐름 설명

### 1. 마이페이지 후기 목록 조회
- 사용자가 마이페이지 "내 후기" 탭 클릭
- 기존 API에 수료증 정보 포함하여 응답

### 2. 수료증 이미지 표시
- 후기 목록에 수료증 썸네일 표시
- 수료증 승인 상태(PENDING/APPROVED/REJECTED) 표시

### 3. 이미지 확대 보기
- 썸네일 클릭 시 원본 이미지 모달로 표시

## 예외 처리

| 조건 | 처리 |
|------|------|
| 인증되지 않은 사용자 | 401 Unauthorized, 로그인 페이지로 리다이렉트 |
| 후기가 없는 경우 | 빈 목록 표시 |

## 관련 API

| Method | Endpoint | 설명 |
|--------|----------|------|
| GET | /api/v1/mypage/reviews | 내 후기 목록 조회 (수료증 정보 포함) |
