## ✅ 정리

<관리자 페이지에서 기관 회원 가입 '승인' 버튼 누르면 기존에 기관 ID가 관리자 ID에서 기관 ID로 바뀌어야 하는 로직에 대하여>

1. 기관 데이터를 다 셋팅해놓고 '승인' 버튼 누르면 기관 회원과 기관 ID를 매핑만 하는 방식으로 구현하자.
2. 이렇게 되면 기관회원이 가입할 때 이미 등록된 기관 데이터 중에서 고를 수 있도록 하는 프론트 화면이 추가로 필요하다.
3. 없는 기관의 경우 추가 요청할 수 있도록 하자.
